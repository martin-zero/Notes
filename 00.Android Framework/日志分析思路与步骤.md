# 蓝牙

`EventParser: processa2dp_sinkOnA2dpSinkConnectionStateChangedEvent` 蓝牙底层a2dp连接状态变化事件

# USB
**关键字:**
`UsbHostManager`  USB连接具体信息
`usb 3-1` USB Kernel日志，usb设备的挂载、卸载日志

# 音视频
### 音频卡顿
**关键字:**
`DMSDP_TRANSPORT_DATASESSION` 媒体流信息，出现underrun表示媒体流出现欠载

### 音频焦点
**关键字:**
`CarAudioFocus` 音频焦点处理

**关键日志:**
获取音频焦点:
```log
CarAudioFocus: AUDIOFOCUS_REQUEST_GRANTED
CarAudioFocus: onAudioFocusGrant
```

释放音频焦点:
```log
CarAudioFocus: onAudioFocusLoss
```

# 休眠、STR
### 通过logcat电源管理服务排查
**关键字:**
`PowerManagerService` 电源管理相关日志
`DisplayPowerController` 屏幕的亮灭的信息

**关键日志**：
进入STR:
```log
PowerManagerService: Sleeping (uid 1000)...
```
正在被唤醒:
```log
PowerManagerService: Waking up from Asleep (uid=1000, reason=WAKE_REASON_APPLICATION, details=android.server.wm:SCREEN_ON_FLAG)...
```

### 通过QNX LaGvm日志定位
**日志路径:** 
`qnx_dump/la_gvm或qnx_log/cycle_xxx_x/LaGvm.txt`(LaGvm文件上的时间+8为Android侧对应时间)

**关键字:**
`suspend` 通过该关键字定位阻止休眠的进程号
