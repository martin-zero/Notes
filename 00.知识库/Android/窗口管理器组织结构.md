1. **ActivityRecord**
    
    - 对应一个 Activity 实例（界面和生命周期信息）
        
    - 最小的显示单元
        
    
2. **Task**
    
    - Activity 的逻辑集合
        
    - 管理 Activity 的启动顺序和 Back 栈
        
    - 一个 Task 内部 Activity 是 **栈结构（LIFO）**
        
    
3. **ActivityStack**
    
    - Task 的容器
        
    - 管理栈内 Task 的显示顺序、焦点和跨 Display 移动
        
    - 一个 Display 可以有多个 ActivityStack


**包含关系：** ActivityStack（Task（ActivityRecord））
