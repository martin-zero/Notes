# 值类别
## 概述
C++11中，值类别可以分为**左值（lvalue）、将亡值（xvalue）以及纯右值（prvalue）**，值类别用来精确描述表达式的属性，从而支持:
- 移动语义（move semantics）
- 完美转发（perfect forwarding）
- 减少不必要拷贝

## 左值(lvalue)
### 定义
**左值是有“持久身份”的对象**，可以取地址，生命周期较长。

### 特征
- 有名字（变量）
- 可以取地址（&）
- 可出现在赋值号左侧

### 示例
```cpp
int a = 10; // a是左值
"hello"; // 字符串常量是左值
++i; // 前置自增是左值
```

## 纯右值(prvalue)
### 定义
临时对象或字面值，没有持久身份。

### 特性
- 无地址（不能 &）
- 生命周期短（通常表达式结束即销毁）
- 用于初始化或计算

### 示例
```cpp
10; // 字面量是纯右值
a + b; // 表达式结果是纯右值
std::string("hi");
i++; // 后置自增是右值
```

## 将亡值(xvalue)
### 定义
即将被销毁，但是资源可以被转移

### 特性
- 有身份
- 可以被资源转移
- 支持 move

### 示例
```cpp
	int&& b = std::move(a); // 被std::move修饰的变量将变为将亡值
```
