# 值类别
C++11中，值类别可以分为**左值、将亡值以及纯右值**，值类别用来精确描述表达式的属性，从而支持:
- 移动语义（move semantics）
- 完美转发（perfect forwarding）
- 减少不必要拷贝

## 区分方法
一般来说通过**有无身份**、**有无稳定的内存地址**来区分:
	左值有身份并且有稳定的内存地址，可以进行取地址操作。**所有变量都是左值**。
	纯右值没有身份并且没有稳定的内存地址，不可以进行取地址操作。**大多数字面量都是右值**。
	将亡值有身份但是没有稳定的内存地址，虽然可以取地址，但资源即将被转移。**一般使用std::move修饰的值被称为将亡值**。

```cpp
int a = 1;
```
a是左值
